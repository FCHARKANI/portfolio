Mes projets

Projet en développement informatique de creation site vitrine commerce

Objectif du projet :

JavaScript : pour les interactions utilisateur simples (menu déroulant, formulaire, sliders, etc.)

EPG Fes : Stage en Conception Modélisation et Développement informatique d'une application de gestion des étudiants inscription et paiement

Objectif du projet :

Le projet consiste à développer une application de gestion des étudiants pour l’École Polytechnique des Génies (EPG) à Fès, intégrant les fonctionnalités de paiement avec impression et de reporting.

Technologies utilisées :

- Langage de programmation : C#
- Environnement de développement : Visual Studio
- Base de données : SQL Server
- Frameworks et bibliothèques : Entity Framework pour l'accès aux données
-iTextSharp pour la génération de PDF
-Microsoft Report Viewer pour les rapports
-Exporter vers Excel avec ClosedXML
